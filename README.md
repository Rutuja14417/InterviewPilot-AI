# 🎯 InterviewPilot AI — Adaptive Engineering Assessor

DataPrep AI is an interactive full-stack interview assistant designed to help students test, practice, and sharpen their core concepts in Artificial Intelligence, Machine Learning, and Data Science. 

Built using Streamlit and powered by advanced reasoning architecture, the agent conducts simulations by asking single technical questions, analyzing candidates' responses dynamically, providing real-time evaluation scores out of 10, and delivering constructive feedback.

## 🚀 Key Features
- **Dynamic Interview Simulation:** Conducts structured, contextual QA sessions.
- **Automated Grading Engine:** Evaluates inputs instantly and provides an evaluation score out of 10.
- **Core Domain Focus:** Covers Supervised/Unsupervised Learning, Deep Learning, Python structures, and SQL.
- **Session State Memory:** Keeps track of conversational history for continuous evaluation loops.

## 📦 Tech Stack
- **Frontend/Backend Framework:** Streamlit (Python)
- **AI Core integration:** Google Generative AI Architecture
